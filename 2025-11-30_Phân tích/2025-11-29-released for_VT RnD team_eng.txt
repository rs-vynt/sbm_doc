TASK 21: Admin Mode (Recommend a good solution, preferably a delivery app)
───────────────────────────────────────────────────────────────

A. Employee Management
   - Function to add and subtract from the number of employees
   - Remove previously registered employees who left the company

B. Employee Information Management
   - For each company's employee information management, the ID and phone number 
     (if the phone number is incorrect, the system will not work) must match to use 
     the paid/free service
   - If the user's new phone number is updated, the system will continue with the new number

C. Monthly Employee Count/Payment
   - Monthly employee count/payment from the start date to the end of each month
   - If 30 days are not enough, the first month will be calculated based on the number of days
   - Payment function - End of each month

D. User Price Adjustment Function
   - Required for email invoices
   - Five days before the end of each month, a bill is automatically generated and sent to:
     * Registered email address
     * Social media
     * Company's main phone number

E. Free Period Notification & Conversion
   - Notification one day prior to the end of the free period (a separate message will be sent)
   - Subsequently, a paid conversion function – individual or all numbers assigned to each company

F. Company Inspection Building Information
   - Information on each company's inspection building
   - Function to collect all inspection databases for that company
   - Assignment to a dealership, attribution function, or sole ownership

G. Payment Failure Handling
   - Immediate access blocking when an inspection company fails to pay

H. Free User Restrictions
   - Web printing restrictions and DB touch functionality for free users

I. Inspector Registration by Dealership
   - Operation will begin after inspectors are registered by dealership (1, 2, 3, 4, 5, etc.)
   - Settlement system established

J. Operation Status by Dealership
   - Operation status by dealership

K. Operation Based on Worker Phone Numbers
   - Operation will be based on individual worker phone numbers

L. Free Version Watermark
   - Free (app)/(web) versions will print the text "(sample)" as a watermark when printing

M. Paid Service Watermark Removal
   - Paid services will remove the watermark when printing from the web
   - In other words, paid services will remove the "(sample)" text when printing
   - This content can be directly designated as paid/free/unprintable in LSMK admin mode

N. Monthly Worker Count
   - Monthly number of workers per company/agency

O. Free/Paid User Distinction & Notification
   - Free/paid user distinction and notification function by subscriber phone number
   - Sort notification content so the administrator can send it in bulk
